# DDD Student Dashboard

## Overview
This project is a performance and engagement dashboard that tracks user activity and provides real-time analytics.

## Features
- Activity tracking (Video, Quiz, Review)
- Points system and gamification
- Real-time updates
- Interactive charts (Chart.js)
- MongoDB data storage
- Session analytics

## Tech Stack
- Frontend: React + TypeScript
- Backend: Node.js + Express
- Database: MongoDB

## How to Run

### Backend
cd backend  
node server.js  

### Frontend
cd frontend  
npm run dev  

## Description
The dashboard helps students track their progress, view analytics, and stay motivated through achievements and points.
